# @quieto/web

React + Vite SPA for creating and visualizing OKLCH color palettes. Provides seed color input, ramp visualization, contrast highlights, export options, and URL-based state sharing.

## Status

This package is scaffolded but not yet implemented. See the [project roadmap](../../README.md#roadmap) for planned features (Epic 3: Web App MVP).

## Planned Features

- Seed color input with live preview
- Ramp visualization with step details
- WCAG contrast highlighting
- CSS and JSON export
- Shareable URL state via `@quieto/engine` serialization

## Development

```bash
# From the monorepo root
npm run dev -w packages/web
```
